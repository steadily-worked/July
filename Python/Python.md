1. list method

list.pop은, 리스트의 마지막 값을 추출하고 리스트에서 없앤다.

ex) planets = ['Mercury', 'Venus', 'Earth', 'Mars', 'Jupiter', 'Saturn', 'Uranus', 'Neptune', 'Pluto'],

planets.pop()을 하면 .. 'Pluto'가 나옴. 그리고 다시  planets를 하면 Pluto가 사라진 Neptune까지의 리스트로 반환.
